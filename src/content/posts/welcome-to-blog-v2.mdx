## Why Blog v2

This milestone introduces a proper MDX post structure so each post can carry rich metadata and render markdown-first content.

## What changed

### Post format

Every post is now an `.mdx` file with YAML frontmatter.

### Routes

- `/posts` lists all posts
- `/posts/[slug]` renders each post detail page

### Sidebar integration

The existing right sidebar now receives post stats and a basic table of contents.

## What's next

Future milestones can add richer syntax highlighting, reading metrics, and cross-post recommendations.
